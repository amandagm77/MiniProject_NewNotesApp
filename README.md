# 📝 New Notes App (React + Vite)

## 📌 Description

A frontend Notes application built with React and Vite that allows users to create, edit, delete, and search notes. This project demonstrates core React concepts such as functional components, props, state management, hooks, and event handling. Notes are persisted using the browser’s local storage, allowing data to remain after page refresh.

---

## ⚡ Quick Start

```
npm install
npm run dev
```


Then open:
http://localhost:5173

---

## 🚀 Features

- Create new notes
- Edit existing notes
- Delete individual notes
- Clear all notes at once
- Search notes in real-time
- Notes counter display
- Persistent storage using localStorage
- Responsive card-based UI layout

## 🧩 User Stories

### ✅ Implemented
- As a user, I want to create a new note so I can store information quickly.
- As a user, I want to edit existing notes so I can update or correct them.
- As a user, I want to delete notes so I can remove unnecessary items.
- As a user, I want to view all notes so I can easily access them.
- As a user, I want to search notes so I can find specific information quickly.
- As a user, I want my notes to persist so I don’t lose them when I refresh the page.
